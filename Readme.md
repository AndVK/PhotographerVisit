# Installation

1.	Install Dependencies `npm install`
2.	Create and initialize file `.env`
3.	Create Database
4.	Run command `npx sequelize-cli db:migrate`
5.	Run command `npx sequelize-cli db:seed:all`
6.	Start project `npm start`
